# PhD Work REM Project

Codex-generated version and my own rewritten version, compare where the Codex
version is useful or problematic, and then reorganise the final version after

large and are copied local data sources. Only code, notes, derived summaries,

## Structure

- `raw data/oxford/`: copied Oxford/OxCGRT raw CSV files. These files are marked
  read-only in the local file system.

- `derived/`: cleaned two-mode country-policy panels generated by the Rmd.
- `outputs/`: model summaries and diagnostic tables generated by the Rmd.
- `figures/`: optional figures.

## covarite 
At the country level, I plan to use GDP per capita, population, democracy score,
and government effectiveness to capture differences in economic development,
state capacity, regime context, and country size. These variables are treated as
relatively stable background attributes, with pre-Covid or baseline values
preferred where possible.

At the time-varying country-day level, I plan to use Covid severity and
vaccination indicators, including confirmed cases, deaths, and vaccination
progress. These variables capture changing pandemic pressure and should be
lagged before entering the model to avoid simultaneity.

At the policy level, I use policy domain or policy type to distinguish between
containment, economic, health, and vaccine-related measures. These variables help
capture differences between policy categories in how easily they can be adopted,
maintained, strengthened, relaxed, or terminated.

At the dyadic country-country level, I use geographic and relational variables
such as distance, contiguity, common official language, and potentially regime or
economic similarity. These variables are used to construct peer exposure between
countries rather than as ordinary country-level controls
